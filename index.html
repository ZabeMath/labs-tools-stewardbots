Wikimedia Toolforge: ~stewardbots project

This is a multi-mantainer project for several IRC bots and related steward tools.

WARNING: We are in the process of updating this site files from the old system we used in Toolserver, expect 404 errors for now.

List of tools

Currently the following tools are avalaible:

  • Elections: Breakdown of the steward elections.
  • Delete: Overview of open speedy deletion requests.
  • Projects: Overview of number of admins/crats/CU/OS on Wikimedia projects.
  • Userrights statistics: Number of users with a specific right per project.
